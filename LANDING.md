# Enterprise RAG Platform — MVP

Knowledge-as-API for Production LLM Systems — MVP launch materials.

This repository contains the minimal assets to launch a 24-hour MVP of the Enterprise RAG Platform: a secure, multi-tenant RAG API that exposes private knowledge as an API.

Key links:
- Product positioning: [enterprise-rag-platform.md](enterprise-rag-platform.md)
- Monetization: [rag-monetization.md](rag-monetization.md)

MVP Highlights
- Quick 1–click signup (JWT)
- Tenant isolation + API keys
- Ingest one document type (Markdown/PDF)
- Hybrid retrieval (vector + keyword)
- Single `POST /query` RAG API returning `{answer,citations,confidence}`
- Manual invoicing for pilots
